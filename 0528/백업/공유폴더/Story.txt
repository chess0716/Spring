웹 프로젝트 구조
Frontend: React를 사용하여 동적이고 반응형 웹 페이지 구현.
Backend: Spring Boot로 RESTful API 제공.
Database: MongoDB (NoSQL) 또는 MySQL (관계형), 데이터 저장 및 조회.
Machine Learning: Python을 사용하여 사용자 데이터 분석 및 스포츠 종목 추천 (로지스틱 회귀, 결정 트리 등).


기능적 구현 기술
메인 페이지: 이미지 선택기 또는 드롭다운을 통한 사용자 정보 수집 및 유형별 스포츠 종목 자동 분류.
매칭 페이지 (커뮤니티): 사용자 간 매칭, 댓글 기능을 통한 소셜 인터랙션 제공.
마이 페이지: 사용자의 활동 기록 (게시글, 리뷰, 예약, 매칭) 관리 및 개인 설정.
로그인 및 회원가입: 카카오 API를 이용한 소셜 로그인 및 일반 로그인 구현,사용자 정보 인증 및 저장 구현
리뷰 페이지: 사용자 리뷰 CRUD 기능, 권한 관리.
예약 페이지: 시간, 날짜, 인원 선택을 통한 예약 기능.
상세 페이지: 선택한 종목에 대한 상세 정보 및 위치 정보(카카오 지도 API 사용), 예약 페이지로의 연결.
데이터 관련 페이지: 빅데이터 시각화를 통해 계절별, 지역별, 나이별, 소득별 선호 스포츠 분석.

파트 분배 및 담당 팀원 기술
Frontend Developer: React 구현, 카카오 지도 API와 소셜 로그인 통합.
Backend Developer: Spring Boot로 API 개발, 보안 및 데이터 처리.
Database Administrator: MongoDB 또는 MySQL 설정, 스키마 설계 및 유지관리.
Data Scientist: Python을 사용한 데이터 분석 및 머신러닝 모델 구현.
UI/UX Designer: 사용자 경험 디자인 및 프론트엔드 지원.

기술스택 총정리
Frontend: React, JavaScript , TypeScript , Mui 라이브러리 , JSX
Backend: Spring Boot, Java
Database: MongoDB, MySQL
Machine Learning: Python, Scikit-learn, Pandas
APIs: 카카오 지도 API, 카카오 로그인 API , 정부 시설 공공 api
Tools: Git, AWS (환경 구성)

환경 설정 및 기초 구현: 프로젝트 환경 설정, 주요 라이브러리 설치 및 기본 API 구현.
기능 개발: 각 기능별 개발 및 통합 테스트.
